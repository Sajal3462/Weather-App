## Introduction

The Weather App is a web application that provides real-time weather information, including temperature, humidity, and wind speed for a specified city.

## Features

- Search for weather information by entering a city name.
- Display of current temperature, humidity, wind speed, and weather icon.
- Error message for invalid city names.
- Continuous updates for live sunrise and sunset times.
- Responsive design for a user-friendly experience.
